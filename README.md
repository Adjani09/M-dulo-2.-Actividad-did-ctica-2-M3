# Simulador de Teoría de Colas (M/M/1) - Banco de Colombia

Este repositorio contiene un modelo de simulación de eventos discretos desarrollado en Python para analizar y optimizar el sistema de atención al cliente en una sucursal del **Banco de Colombia**. 
El sistema modela 3 cajeros independientes bajo la metodología de colas $M/M/1$ con arribos y tiempos de servicio exponenciales.
